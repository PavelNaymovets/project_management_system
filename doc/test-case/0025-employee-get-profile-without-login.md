# 25: Получение профиля сотрудника по login без указания login

## Описание

Получение карточки сотрудника по login без указания login.

## Предусловия

Администратор авторизован в системе.

## Шаги в postman

* Зайти в коллекцию `projectMenegementSystem`
* Перейти в папку `Employee`
* Выбрать запрос `GetByLogin`
* Нажать `Send`

Выполнится HTTP запрос с методом `GET` на endpoint `http://localhost:8080/pms/api/v1/employee/login`.

## Ожидаемый результат

Вернется ответ со статус кодом `400` и сообщением плохой запрос.

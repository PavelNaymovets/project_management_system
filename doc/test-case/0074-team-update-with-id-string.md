## Описание

Изменение полей команды с указанием id строкового значения

## Предусловия

Команда с атрибутами:

* project - {
  id - 1
  }

, есть в базе данных. Менеджер авторизован в системе.

## Шаги в postman

* Зайти в коллекцию `projectMenegementSystem`
* Перейти в папку `Team`
* Выбрать запрос `UpdateTeam`
* Нажать `Send`

Выполнится HTTP запрос с методом `PUT` на endpoint `http://localhost:8080/pms/api/v1/project`. Тело запроса:

```
{
    "id": dfsdfsdf,
    "project": {
        "id": 4
    } 
}
```

## Ожидаемый результат

Вернется ответ со статус кодом `400` и сообщение плохой запрос.

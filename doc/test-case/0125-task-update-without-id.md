# 125: Изменение полей задачи без указания id

|    Автор    |    Дата    | Средства тестирования | Результат |
|:-----------:|:----------:|:---------------------:|:---------:|
| Наумовец П. | 2023-06-15 |   Postman v10.14.9    |  Прошел   |

## Описание

При редактировании полей задачи должно указываться поле id.

## Предусловия

Задача с атрибутами:

* id - 6,
* name - новая задача,
* description - описание новой задачи,
* laborCosts - 10,
* deadline - 2023-05-24,
* status - новая

, есть в базе данных. Пользователь авторизован в системе.

## Шаги в postman

* Зайти в коллекцию `projectMenegementSystem`
* Перейти в папку `Task`
* Выбрать запрос `UpdateTask`
* Нажать `Send`

Выполнится HTTP запрос с методом `PUT` на endpoint `http://localhost:8080/pms/api/v1/task`. Варианты тела запроса:

```
{
    "name": "Обновленная задача",
    "description": null,
    "project": {
        "id": 1
    },
    "laborCosts": "11",
    "deadline": "2024-05-24"
}
```

## Ожидаемый результат

Вернется ответ со статус кодом `400` и сообщением, что id не указано.

## Фактический результат

```
{
    "statusCode": 400,
    "message": "Field id not specified."
}
```

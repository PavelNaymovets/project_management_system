# 80: Удаление команды с указанием id = null

|    Автор    |    Дата    | Средства тестирования | Результат |
|:-----------:|:----------:|:---------------------:|:---------:|
| Наумовец П. | 2023-06-14 |   Postman v10.14.9    |  Прошел   |

## Описание

Удалить команду с указанием id = null нельзя.

## Предусловия

Менеджер с правами администратора авторизован в системе.

## Шаги в postman

* Зайти в коллекцию `projectMenegementSystem`
* Перейти в папку `Team`
* Выбрать запрос `DeleteById`
* Нажать `Send`

Выполнится HTTP запрос с методом `DELETE` на endpoint `http://localhost:8080/pms/api/v1/team/null`.

## Ожидаемый результат

Вернется ответ со статус кодом `400` и сообщением плохой запрос
